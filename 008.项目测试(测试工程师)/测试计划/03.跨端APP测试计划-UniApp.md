## 一、项目概述

### 1.1 技术栈

| 维度 | 实际情况 |
| --- | --- |
| 框架 | uni-app 3.x（alpha 预发布版） |
| Vue 版本 | **Vue 3**（createSSRApp） |
| 语言 | **TypeScript（strict 模式）** |
| 状态管理 | **Vuex 4**（非 Pinia） |
| UI 组件库 | **无第三方库**（纯原生组件 + 自研 CSS） |
| HTTP | 自研封装，基于 `uni.request`（`src/utils/api.ts`） |
| 构建 | Vite 5.2.8 + `@dcloudio/vite-plugin-uni`（**CLI 方式**，非 HBuilderX 工程） |
| 图表 | 原生手写 SVG（趋势图/饼图） |
| 目标平台 | H5、App(iOS/Android)、微信/支付宝/百度/QQ/头条/小红书等小程序（脚本全平台） |
| 类型检查 | `npm run type-check`（vue-tsc） |

### 1.2 目录结构

```
uniapp-projct/
├── package.json / vite.config.ts / tsconfig.json / manifest.json / pages.json
└── src/
    ├── main.ts / App.vue        # 入口（onLaunch 触发 checkAuth）
    ├── pages/                   # 7 页：login/register/index/records/food/statistics/profile
    ├── api/                     # transaction/category/account/food/statistics
    ├── store/index.ts           # Vuex（token/user/isAuthenticated + auth actions）
    ├── utils/api.ts             # uni.request 封装 + token 存取
    ├── types/index.ts           # TS 类型
    └── styles/design-system.css
```

**缺失目录**（已知）：`src/static/`（tabBar 图标引用但不存在）、`src/components/`（无复用组件）。

### 1.3 启动方式

| 命令 | 作用 |
| --- | --- |
| `npm run dev:h5` | H5 开发（端口 3000） |
| `npm run dev:mp-weixin` | 微信小程序（配合微信开发者工具） |
| `npm run dev:app(-android/-ios)` | App |
| `npm run build:*` | 对应平台构建 |
| `npm run type-check` | vue-tsc 类型检查 |

- **接口地址硬编码**：`src/utils/api.ts:10` → `http://localhost:8081/api`，**无 .env、无 dev/prod 区分**。
- **微信小程序**：`manifest.json` 中 `mp-weixin.appid` 为空，仅可本地预览，无法上传/真机审核。
- 超时 15s。

## 二、测试目标与范围

### 2.1 目标

1. 验证 **H5 端**各功能模块可用（本轮主战场）。
2. **重点验证跨端兼容性**，暴露小程序/App 端的已知大面积不可用问题。
3. 验证**已知功能性缺陷**（统计页崩溃、tabBar 图标缺失、rememberMe、餐饮筛选等）。
4. 验证鉴权链路（Token 存储、登录守卫）。

### 2.2 范围内页面

| 页面 | tabBar | 模块 |
| --- | :---: | --- |
| `pages/login` | — | 登录 |
| `pages/register` | — | 注册 |
| `pages/index` | ✅ 首页 | 快捷记账 |
| `pages/records` | ✅ 交易 | 交易记录 |
| `pages/food` | ✅ 餐饮 | 餐饮明细 |
| `pages/statistics` | ✅ 统计 | 统计分析 |
| `pages/profile` | ✅ 我的 | 设置/分类管理 |

### 2.3 范围外 / 约束

- 真实支付、微信/手机号登录、短信验证码（产品未实现）。
- 微信小程序**真机审核/上传**（appid 为空，仅本地预览）。
- 小程序/App 端因兼容问题预期大面积不可用——**作为缺陷集中评估，而非逐功能深测**。
